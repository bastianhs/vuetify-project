# Cara run

1. Copy file .env.example ke file .env di directory yang sama

2. Run docker

   ```sh
   docker compose down
   docker compose up --build
   ```

3. Contoh Frontend

   ```sh
   http://localhost:3000
   ```

4. Contoh Backend

   ```sh
   http://localhost:8080/api/users
   ```

   ```sh
   http://localhost:8080/api/users/1
   ```
